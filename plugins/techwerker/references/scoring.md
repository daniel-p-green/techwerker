# Scoring Notes

Keep scoring legible and easy to override:

- Reward matching topics, priority hosts, preferred neighborhoods, open links, and preferred time windows.
- Penalize excluded formats, avoided clusters, missing links, and awkward time windows.
- Bucket by day and time slot before ranking the portfolio.
- Preserve geographic diversity inside a bucket, but do not scatter the day across impossible locations.
- Oversignup defaults to 3 plausible applications per intended slot.

The user can edit `preferences.json` directly when the ranking feels wrong.
